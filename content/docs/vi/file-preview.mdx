# Preview tệp

VercelDrive xem trước các loại tệp OneDrive phổ biến ngay trong trình duyệt.

Nhóm preview hỗ trợ gồm:

- Markdown và văn bản thuần
- Mã nguồn
- Hình ảnh
- Âm thanh
- Video
- PDF
- Tài liệu Office khi trình duyệt hỗ trợ nhúng
- EPUB
- File URL shortcut

Khi không thể preview một loại tệp, VercelDrive hiển thị card mặc định kèm thao tác tải xuống.

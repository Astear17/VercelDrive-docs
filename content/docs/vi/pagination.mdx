# Tự động tải thêm (Auto-fetch)

OneDrive trả về thư mục lớn theo từng trang, mỗi trang tối đa 200 mục. VercelDrive tự động tải tất cả các trang theo thứ tự — người dùng không cần nhấn nút để tải thêm tệp.

Khi thư mục chứa nhiều mục hơn một phản hồi API, VercelDrive theo liên kết phân trang API và tự động tải trang tiếp theo. Chỉ báo tiến trình xuất hiện ở cuối danh sách thư mục khi các trang bổ sung đang được tải.

OneDrive API vẫn giới hạn mỗi trang ở 200 mục.

# Auto-fetch

OneDrive returns large folders in pages of up to 200 items. VercelDrive automatically fetches all pages in sequence — visitors do not need to click a button to load more files.

When the folder contains more items than a single API response, VercelDrive follows the API pagination link and loads the next page automatically. A progress indicator appears in the folder footer while additional pages are loading.

The OneDrive API still limits each page to 200 items.

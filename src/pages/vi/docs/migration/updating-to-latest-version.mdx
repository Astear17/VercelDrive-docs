# Cập nhật lên phiên bản mới nhất

Về cơ bản, bạn đang đồng bộ một fork của kho lưu trữ để giữ kho của bạn cập nhật với kho gốc (upstream). Xem thêm [GitHub - Syncing a fork](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/working-with-forks/syncing-a-fork).

## Từ giao diện Web

- Trên GitHub, mở trang chính của kho đã fork muốn đồng bộ với kho gốc.
- Chọn menu thả xuống **Fetch upstream**.
- Xem lại chi tiết các commit từ kho gốc, rồi bấm **Fetch and merge**.
- Nếu thay đổi từ kho gốc gây xung đột, GitHub sẽ yêu cầu bạn tạo pull request để giải quyết.

## Dòng lệnh

Clone kho đã fork về máy cục bộ và:

```bash
# Lấy thay đổi từ upstream
git fetch upstream

# Chuyển sang nhánh mặc định - main
git checkout main

# Gộp thay đổi từ upstream vào fork
git merge upstream/main

# Nếu có xung đột - giải quyết thủ công tại chỗ
# Commit và đẩy lên GitHub
git commit -m "xxx"
git push
```

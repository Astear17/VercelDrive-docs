# What's New In VercelDrive

This page tracks changes in the current VercelDrive fork.

## 2026-06-20

### Added
- Security page documenting authentication, upload password, protected folders, and OAuth token management
- Troubleshooting hub with common issues and fixes
- API fetch error section in deployment error FAQ
- Auto-fetch documentation replacing old manual pagination description
- Grid view masonry layout documentation
- File preview showcase with per-category screenshots

### Changed
- Getting Started now includes access mode explanations for read-only, read/write, and protected folders
- Pagination page renamed to Auto-fetch to reflect current behavior
- Grid view page updated to describe masonry layout that handles mixed aspect ratios
- Sponsor page reframed as Support with useful project links

### Removed
- Stale upstream discussion links from search and identity verification pages

### Docs
- Bilingual English/Vietnamese coverage for all new and updated pages
- Screenshot placeholders added for visual features

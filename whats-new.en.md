# What's New In VercelDrive

This page tracks the current VercelDrive fork rather than the archived upstream project.

## Current Documentation Refresh

- Getting Started now starts with a read-only versus read/write deployment choice.
- Vercel deployment buttons include variants for uploads, protected routes, Redis prefixes, contact email, and upload conflict behavior.
- Advanced setup now documents Microsoft Graph read-only and read/write permission paths.
- Upload documentation covers `UPLOAD_PASSWORD`, upload sessions, large files, folder uploads, and Live Photo component uploads.
- Grid view, pagination, protected folders, direct links, and multi-file download pages were updated for the current VercelDrive behavior.
- Vietnamese routes are supported in the docs with `/vi/...`; `/vi-VN/...` redirects to `/vi/...`.

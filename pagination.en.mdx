# Pagination

OneDrive returns large folders in pages. VercelDrive automatically follows the API pagination in the background and appends each page as it arrives, so visitors do not need to click through every 200 items manually.

The OneDrive API still limits each page to 200 items. While the next page is loading, VercelDrive shows progress in the folder footer.
